假设一个代码块有两个输入A,B, A的范围是1..32,B的范围是1..8
那么生成输入数据的范围开放先后顺序为
1. 顶点A[1,32] B[1,8]
2. 一级分型A[16] B[4]
3. 二级分型A[8,24] B[2,6]
4. 三级分型A[4,12,20,28] B[1,3,5,7]
5. 四级分型A[2,6,10,14,18,22,26,30]
6 ...
在上一级采集完后再采集下一级, 同级间可随机顺序
优先级示例: 

  1 2 3 4 5 6 7 8 9
1 1   3   2   3   1
2 3   3   3   3   3
3 2   3   2   3   2
4 3   3   3   3   3
5 1   3   2   3   1

取样数量上限: 可对每个输入设置不同的取样数量上限
如输入范围1..1000取样数量上限设置为12时
顶点(2): 1, 1000
一级(1): 500
二级(2): 250,750
三级(4): 125,375,625,875
四级(3): 8个中随机取3个
